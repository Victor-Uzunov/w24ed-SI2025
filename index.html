Help Guide

Managing Programmes

  • Click "New Programme" to create a programme
  • Fill in the required details (name, years, type)
  • Click on a programme to view its courses
  • Use the delete button to remove a programme

Managing Courses

  • Select a programme first
  • Click "New Course" to add a course
  • Fill in course details
  • Select prerequisites from earlier years
  • Use the delete button to remove a course
